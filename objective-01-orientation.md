# Objective 1: Holiday Hack Orientation

## Overview
The Holiday Hack Orientation objective is a warm-up exercise designed to introduce participants to the Holiday Hack terminal interface and challenge structure. It focuses on familiarization rather than technical difficulty.

## Environment
The interface contains:
- an **upper input pane** for commands
- a **lower output pane** for responses

## Task
Participants are instructed to:
- type a specific command or phrase exactly as instructed
- submit it in the terminal input area
## Solution: 
Typing the word "answer" into the top of the terminal will mark this complete.
## Result
Once the correct command is entered:
- the system validates input
- the objective is marked as complete

## Key Learning
- understanding the challenge platform
- preparing for later objectives
